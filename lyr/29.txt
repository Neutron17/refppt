Mostan, ti, hatalmasak, / Tekintélyes nagy urak, / Adjatok az Istennek, / Dicsőséget nevének! / Mint hatalmas Istenteket / Féljétek, tisztelvén őtet! / Templomában őt áldjátok, / És térdet, fejet hajtsatok!

Az Úr szava megzendül, / A vizeken megdördül; / Mennydörgő dicsősége / Elhat a nagy tengerre. / Az Úrnak rettentő szava / Nagy hatalmát megmutatja. / Az Úrnak dördülő szaván / Nagy volta meglátszik nyilván.

De az ő templomában / Ő hívei mindnyájan / Hirdetik nagy erejét, / Beszélik dicsőségét. / Az Úr ült az özönvízen, / Mint bíró ítélőszéken; / Az Úrnak ő királysága, / Örökké megáll országa.

Az Úr az ő népének, / Erőt ád seregének, / Őket híven megtartja, / Békességgel megáldja.
